In this assignment, I created a basic Node.js server using the built-in http module. The server listens on a port and responds based on the URL entered. 

For example:
/ → Home Page

/about → About Page

/contact → Contact Page

If the user visits an unknown route, it shows a “404 Page Not Found” message. 
I used res.write() and res.end() to send responses.This project shows how Node.js can handle simple routing and serve web pages without needing frameworks like Express.
